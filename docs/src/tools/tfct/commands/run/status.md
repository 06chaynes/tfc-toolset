# status

## Description

Get the status of a run

## Usage

```bash
tfct run status [options]
```

## Options

| Short | Long                | Description        |
| ----- | ------------------- | ------------------ |
| `-i`  | `--run-id <RUN_ID>` | The id of the run. |

## Examples

### Get the status of a run

```bash
tfct run status --run-id "run-id"
```
