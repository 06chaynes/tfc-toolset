# run

## Description

Manage runs.

## Usage

```bash
tfct run [command] [options]
```

## Subcommands

| Name                    | Description                   |
|-------------------------|-------------------------------|
| [`status`](./status.md) | Get the status of a run.      |
| [`plan`](./plan.md)     | Queue up plan only runs.      |
| [`apply`](./apply.md)   | Queue up plan and apply runs. |
| `help`                  | Prints help information.      |
