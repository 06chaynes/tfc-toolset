# cancel

## Description

Cancel a run

## Usage

```bash
tfct run cancel [options]
```

## Options

| Short | Long                | Description        |
| ----- | ------------------- | ------------------ |
| `-i`  | `--run-id <RUN_ID>` | The id of the run. |

## Examples

### Cancel a run

```bash
tfct run cancel --run-id "run-id"
```
