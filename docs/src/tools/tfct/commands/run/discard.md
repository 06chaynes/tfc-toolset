# discard

## Description

Discard a run

## Usage

```bash
tfct run discard [options]
```

## Options

| Short | Long                | Description        |
| ----- | ------------------- | ------------------ |
| `-i`  | `--run-id <RUN_ID>` | The id of the run. |

## Examples

### Discard a run

```bash
tfct run discard --run-id "run-id"
```
